# 🛍️ ادمین پنل مدیریت فروشگاه

این یک **ادمین پنل کاربردی، زیبا و منحصر‌به‌فرد** است که کار مدیریت فروشگاه را برای مدیران محترم آسان‌تر می‌کند. با این پنل، مدیران می‌توانند با راحتی و دقت بیشتری بر وب‌سایت خود کنترل داشته باشند.

> **نمونه موردی:** این پروژه برای فروشگاه خیالی "شیراز پیسی" طراحی شده، اما برای مدیریت هر نوع فروشگاه دیگری نیز قابل استفاده و دلچسب است.

⚠️ **توجه:** این پروژه در حال حاضر **صرفاً در سمت کاربر (فرانت‌اند)** توسعه یافته و هنوز به بک‌اند متصل نشده است.

## ✨ ویژگی‌ها

- 📊 **نمودارهای تحلیلی:** نمایش آمار و بررسی کلی فروشگاه با استفاده از کتابخانه Chart.js
- 🔍 **جست‌وجوی پیشرفته:** جست‌و‌جو در محصولات و داده‌ها بر اساس نیاز
- 🎯 **فیلترهای هوشمند:** فیلتر محصولات بر اساس دسته‌بندی، قیمت و...
- 📦 **داده‌های شبیه‌سازی‌شده:** تمام اطلاعات جدول‌ها از فایل JSON خوانده می‌شوند تا ارتباط با سرور شبیه‌سازی شود
- 📱 **طراحی واکنش‌گرا:** نمایش عالی در تمام دستگاه‌ها
